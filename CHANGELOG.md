# 変更履歴

このファイルは、AI Story Builderプロジェクトの重要な変更を記録します。


## [1.0.0]

### 追加
- 初回リリース
- 複数AIプロバイダー対応（OpenAI、Claude、Gemini、ローカルLLM）
- キャラクター管理システム
- プロット構築ツール
- あらすじ自動生成
- 章立て構成機能
- 執筆支援機能
- ダークモード対応
- レスポンシブデザイン
- PWA対応
- オフライン機能
- セキュリティ機能（APIキー暗号化、入力値サニタイゼーション）
- パフォーマンス最適化
- 複数デプロイメントプラットフォーム対応

### 技術仕様
- React 18.3.1
- TypeScript 5.5.3
- Vite 7.1.3
- Tailwind CSS 3.4.1
- Dexie 4.2.0（ローカルデータベース）
