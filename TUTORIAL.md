# AI Story Builder 完全チュートリアル 〜AIと二人三脚で挑む小説執筆〜

## はじめに

AI Story Builderは、プロットの構想から本文の執筆、最終的な推敲に至るまで、AIが専属編集者のように寄り添う小説創作支援ツールです。
本チュートリアルでは、**「小説を書きたいけれど、途中で筆が止まってしまう」「AIをどう使えばいいかわからない」**という課題を解決するため、ステップ1（物語の種）からステップ8（エクスポート）までの具体的な機能と、AIの最適な活用方法を解説します。

サイドバーの「制作ワークフロー」には、大きく **構想** → **執筆** → **仕上げ** の3フェーズ・8ステップが並んでいます。
また、画面右側の **ツールサイドバー** には、どのステップからでも使える共通ツール群（用語集・相関図・タイムライン・世界観・伏線トラッカー・感情マップ・イメージボード・クイックメモ・AIチャット相談）が常備されています。

## ステップ別チュートリアルとAI活用術

---

### 【構想フェーズ】

---

#### ステップ1：物語の種（プロット基本設定）

**目的**: 物語の骨格となる6つの基本要素を設計し、作品全体の方向性を決めます。

このステップでは、以下の6つのフィールドを入力して物語の基盤を作ります。

| フィールド | 説明 | 文字数上限 |
|---|---|---|
| **メインテーマ** | 物語の核心（例：友情と成長） | 100文字 |
| **舞台設定** | 世界観・時代・場所 | 300文字 |
| **物語の引き（冒頭の魅力）** | 読者を引き込む導入の要素 | 300文字 |
| **主人公の目標** | 主人公が達成したいこと | 100文字 |
| **主要な障害** | 目標を阻む困難 | 100文字 |
| **物語の結末** | 物語がどう終わるか | 200文字 |

**💡 AI活用術**:

- **フィールド単位のAI提案**: 各フィールドの横にある「AI提案」ボタンを押すと、プロジェクト情報（タイトル・ジャンル・キャラクター設定など）を踏まえた提案をAIが生成してくれます。既に入力済みの他フィールドとの一貫性も考慮されます。
- **サンプル例の参照**: 各フィールドのメニューから「サンプルを表示」を選ぶと、テンプレート文例が表示されます。ゼロから書くのが難しい場合の出発点として最適です。
- **AIチャット相談**: 右サイドバーの「AIチャット相談」ボタンから、漠然としたアイデアをAIに投げかけて壁打ちすることができます。「こんな話を書きたいんだけど、どんなテーマがいい？」といった相談が可能です。

---

#### ステップ2：キャラクター

**目的**: 登場人物のプロフィール（名前・役割・外見・性格・背景）を作成し、物語の中で彼らがどう動くかを定義します。

**💡 AI活用術**:

- **自動プロフィール生成**: 右サイドバーの「支援」タブからキャラクターを一括自動生成したり、不足しているキャラクターの提案を受けることができます。
- **画像分析からの外見描写生成**: キャラクターカードの画像アップロード機能で立ち絵やイメージ画像をアップロードし、AIが画像から外見描写文を自動生成します（Cloud AI使用時）。
- **キャラクター憑依モード**: キャラクターカードの 🎭 ボタンから、設定したキャラクターにAIを「憑依」させて直接会話できます。「この状況ならどう返す？」をシミュレーションすることで、そのキャラ特有の口調や行動原理を掴むことができます。
- **相関図ツール**: 右サイドバーのツールタブから「相関図」を開くと、キャラクター間の関係性を視覚的に管理できます。

---

#### ステップ3：構成（プロット構成詳細）

**目的**: ステップ1で設定した基本要素をベースに、物語の構造フレームワーク（起承転結・三幕構成・四幕構成など）を選び、各パートの詳細を肉付けします。

**💡 AI活用術**:

- **フレームワーク自動適用**: 「起承転結」「三幕構成」「四幕構成」などの構造テンプレートを選択し、ステップ1の設定を元にAIが各パートの内容を提案します。右サイドバーの「支援」タブから構造に合わせたAI提案を利用できます。

---

### 【執筆フェーズ】

---

#### ステップ4：あらすじ

**目的**: これまでに作成した設定とプロットを統合し、作品の全体像を俯瞰できる「あらすじ」を完成させます。目標は約500文字です。

**💡 AI活用術**:

- **あらすじの自動生成**: 右サイドバーの「支援」タブから、プロット情報・キャラクター情報を統合してAIにあらすじを生成させることができます。矛盾点があればこの段階で整理できます。
- **手動編集との併用**: AIが生成した下書きをベースに、自分の言葉で磨き上げることで、より作品に合ったあらすじに仕上げましょう。

---

#### ステップ5：章立て

**目的**: 物語を「章（チャプター）」に分割し、各章のタイトル・要約・登場キャラクター・舞台・雰囲気・重要イベントを定義します。

**💡 AI活用術**:

- **章の自動生成**: 右サイドバーの「支援」タブから、プロット構成に基づいてAIが各章の概要を自動生成します。
- **章の強化（エンハンス）**: 各章のカードにある強化ボタンから、AIが内容の充実化・分割・深掘りなどの提案を行います。
- **章の分割**: 内容が多い章を、AIの提案を元に複数の章に分割できます。
- **ドラッグ＆ドロップ**: 章の順番はドラッグ＆ドロップで自由に入れ替え可能です。
- **伏線トラッカーとの連動**: 右サイドバーのツールタブから「伏線トラッカー」を開くと、章単位で伏線の設置・ヒント・回収ポイントを管理できます。どの章で伏線を張り、どの章で回収するかを整理しながら章立てを進めましょう。

---

#### ステップ6：執筆（ドラフト）

**目的**: 章ごとに、実際の本文を執筆します。縦書きエディタや禅モードで集中力を高めながら書けます。

**💡 AI活用術**:

- **「続きを書く」アシスト**: 右サイドバーの「支援」タブにある「続きを書く」ボタンで、直前の文脈をAIが読み取り、次に続く文章を提案してくれます。筆が止まったときの強力な味方です。
- **リライト案の生成**: 「リライト案」タブで、執筆中のテキストの代替表現をAIに提案させることができます。別の文体や描写スタイルを試したいときに便利です。
- **トーン変更**: 「トーン変更」タブで、文章の雰囲気（シリアス・コミカルなど）を変える提案を受けられます。
- **縦書きモード**: 表示設定パネルから縦書き表示に切り替え可能。日本語小説の雰囲気を確認しながら執筆できます。
- **禅モード**: 表示設定パネルの「禅モード」ボタンで、UI要素を最小限にしたフルスクリーンの執筆専用画面に切り替わります。
- **伏線パネル**: 執筆画面内に伏線管理パネルが組み込まれており、執筆中に伏線の状態を確認・更新できます。

---

### 【仕上げフェーズ】

---

#### ステップ7：分析（作品評価 / Review）

**目的**: 完成した原稿をAIに多角的に評価させ、作品の強みと改善点を明確にします。

このステップでは、4つの評価モードと4段階の厳しさレベルを組み合わせて分析を行います。

| 評価モード | 分析内容 |
|---|---|
| **構造・プロット** | 物語の構成、一貫性、ペース配分を分析 |
| **キャラクター** | キャラクターの動機、成長、独自性を評価 |
| **文体・表現** | 文章の読みやすさ、描写力、五感表現をチェック |
| **読者ペルソナ** | 想定読者になりきって感想と市場性を評価 |

| 厳しさレベル | 特徴 |
|---|---|
| やさしい | 良い点を重視し、建設的なフィードバックを提供 |
| 普通 | バランスの取れた評価（デフォルト） |
| 厳しい | より厳格な基準で評価し、改善点を明確に指摘 |
| 辛辣 | プロの編集者として厳しく評価し、問題点を率直に指摘 |

**💡 AI活用術**:

- **評価対象の柔軟な選択**: あらすじ・特定の章・作品全体（全章結合）・カスタムテキスト・外部ファイル（.txt/.md）のいずれかを評価対象として選べます。
- **評価結果の保存とエクスポート**: 評価結果はプロジェクト内に保存でき、マークダウン形式でダウンロードも可能です。過去の評価との比較で成長を確認しましょう。
- **感情マップ**: 右サイドバーのツールタブにある「感情マップ」を使えば、シーンごとの感情の流れをグラフで可視化して確認できます。

---

#### ステップ8：エクスポート

**目的**: 完成した作品や設定資料を、指定した形式で出力します。

| 出力形式 | 説明 |
|---|---|
| **テキスト (.txt)** | シンプルなテキスト形式 |
| **マークダウン (.md)** | 構造化されたマークダウン形式 |
| **HTML (.html)** | Webブラウザで表示可能なHTML形式 |

**主な機能**:

- **エクスポート内容の選択**: 基本情報・キャラクター・プロット・あらすじ・章立て・イメージボード・草案・用語集・相関図・タイムライン・世界観設定・伏線トラッカー・クイックメモの中から、出力する項目を自由に選択できます。
- **プリセット**: 「完全版」（すべて含む）・「草案のみ」・「設定資料のみ」（草案・メモを除く）の3つのプリセットで素早くオプションを切り替えられます。
- **ファイル名のカスタマイズ**: タイムスタンプやバージョン番号の付与が可能です。
- **プレビュー**: エクスポート前に出力内容をプレビューし、セクション検索でジャンプできます。
- **クリップボードコピー**: ファイル出力の代わりに、内容全体をクリップボードにコピーすることもできます。

## 全ステップ共通のツール（ツールサイドバー）

画面右側のツールサイドバーには、どのステップからでもアクセスできる共通ツールが用意されています。

| ツール | 説明 |
|---|---|
| **イメージボード** | 参考画像やキャラクターイラストを整理するギャラリー |
| **用語集** | 固有名詞や独自設定をデータベースとして管理 |
| **相関図** | キャラクター間の関係性を視覚的に表示・管理 |
| **タイムライン** | 物語内の出来事を年表形式で管理 |
| **世界観** | 世界の独自ルールや設定をカテゴリ別に記録 |
| **伏線トラッカー** | 伏線の設置・進行・回収状態をリスト管理 |
| **感情マップ** | シーンごとの感情変化をグラフで可視化 |
| **クイックメモ** | 3つのタブで自由にメモを管理（アイデア・タスク・ノート） |
| **AIチャット相談** | 創作上の悩みや設定の相談をAIに自由に行える汎用チャット |

## さいごに

AI Story Builderは、あなたの創造性を奪うものではなく、**「あなたの書きたいものを、最後まで書き切るための伴走者」**です。
設定の矛盾に気づいてくれない孤独な執筆から抜け出し、最新のAIと一緒に最高の物語を創り上げてください。
