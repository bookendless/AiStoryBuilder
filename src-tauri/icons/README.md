# Tauri Icons

このディレクトリには以下のアイコンファイルを配置してください：

- `32x32.png` - 32x32ピクセルのPNGアイコン
- `128x128.png` - 128x128ピクセルのPNGアイコン  
- `128x128@2x.png` - 256x256ピクセルのPNGアイコン（高DPI用）
- `icon.icns` - macOS用アイコンファイル
- `icon.ico` - Windows用アイコンファイル

## アイコン生成方法

1. 1024x1024ピクセルの正方形の画像を用意
2. 以下のツールでアイコンを生成：
   - [Tauri Icon Generator](https://tauri.app/v1/guides/features/icons/)
   - [App Icon Generator](https://appicon.co/)

## 注意事項

- アイコンは正方形である必要があります
- 背景は透明または単色を推奨します
- ファイル名は上記と完全に一致させる必要があります
